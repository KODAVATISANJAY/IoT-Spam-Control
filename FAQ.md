# FAQ - IoT Spam Control

## General Questions

### Q: What is IoT Spam Control?
A: A system that detects and blocks spam traffic in IoT networks using DSA and ML.

### Q: How accurate is the spam detection?
A: The system achieves 95%+ accuracy with the trained ML model.

### Q: What IoT protocols are supported?
A: MQTT, CoAP, HTTP, and TCP/IP.

## Technical Questions

### Q: What libraries are required?
A: scikit-learn, numpy, pandas, matplotlib for Python.

### Q: How do I train the ML model?
A: Run python train_model.py with your dataset.

### Q: Can I use it with Raspberry Pi?
A: Yes, it runs on any device with Python 3.8+.
