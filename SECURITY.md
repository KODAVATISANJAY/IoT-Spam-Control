# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in the IoT Spam Control project, please report it by creating an issue on this repository.

## Security Measures

- Input validation on all user inputs
- Secure file handling
- Environment variable configuration
- Regular dependency updates
